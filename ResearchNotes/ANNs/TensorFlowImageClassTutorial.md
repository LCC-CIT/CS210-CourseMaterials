Notes on: [Basic image classification](https://www.tensorflow.org/tutorials/keras/classification)

Intro to Keras: [Keras: The high-level API for TensorFlow](https://www.tensorflow.org/guide/keras)

Using TF version 2.19.0

Uses this dataset: [Fashion MNIST](https://github.com/zalandoresearch/fashion-mnist) dataset which contains 70,000 grayscale images in 10 categories. The images show individual articles of clothing at low resolution (28 by 28 pixels)

The original [MNIST dataset](https://en.wikipedia.org/wiki/MNIST_database#MNIST) (Modified National Institute of Standards and Technology database) is a large database of handwritten digits.
