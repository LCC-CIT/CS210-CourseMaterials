About CS210: Introduction to Artificial Intelligence Programming

This course provides an overview of the field of Artificial Intelligence (AI) and how to write computer programs that use AI. Topics include symbolic AI, machine learning, neural networks, generative AI, LLMs, and ethical issues. See the syllabus for details.
